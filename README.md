# Deadline Guardian

## 📌 Project Overview
Deadline Guardian is an AI-powered task management web application that helps users organize tasks, track deadlines, and improve productivity. The application uses Gemini AI to provide intelligent assistance for managing deadlines efficiently.

## ✨ Features
- Add, edit, and delete tasks
- Set deadlines for tasks
- AI-powered task suggestions using Gemini AI
- Responsive and user-friendly interface
- Built with modern web technologies

## 🛠️ Technologies Used
- Next.js
- React
- TypeScript
- Gemini AI API
- CSS

## 🚀 Installation

1. Clone the repository:
```bash
git clone https://github.com/Lamiya09/deadline-guardian.git
```

2. Install dependencies:
```bash
npm install
```

3. Run the project:
```bash
npm run dev
```

4. Open your browser and visit:
```
http://localhost:3000
```
